# math-interpreter
A math interpreter made in C

## Supported operations
 - Addition ( "<b>+</b>" )
 - Subtraction ( "<b>-</b>" )
 - Multiplication ( "<b>*</b>" )
 - Division ( "<b>/</b>" )
 
You can combine these operations using parenthesis ( "<b>(</b>" and "<b>)</b>" )

## Compile and Run

1. compile the project with: `make`
2. run with `./bin/math`
